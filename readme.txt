简介：
   该工程基于MCU MC20P7012B 开发，MCU拥有4K byte ROM以及208 byte RAM，输出文件为*.s19文件，目前已使用3.9k以上。
项目特点：
1、应用于简单的4.2V锂电池充电，同时兼容2节或2节锂电池串联充电
2、具有平衡充管理功能，使充电完成后各节电池电量一致，避免因充电不平衡，导致放电也不平衡，从而电池损坏的问题
3、具有低压，过压，短路，断路保护功能
4、具有低压预充功能，按照锂电池正常充电流程充电
